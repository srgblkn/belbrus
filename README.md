# Бот для транслитерации ФИО

Бот транслитерирует ФИО с кириллицы на латиницу согласно Приказу МИД России от 12.02.2020 № 2113

## Запуск

1. Получите токен у @BotFather в Telegram
2. Клонируйте репозиторий:
```bash
git clone https://github.com/#ваш_логин#/belbrus.git
cd belbrus

## Запустите через терминал
docker run -it -e TOKEN='ВАШ_ТОКЕН' belbrus
